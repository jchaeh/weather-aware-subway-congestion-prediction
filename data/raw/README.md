# Raw Data Sources

본 프로젝트에서 사용한 원본 데이터는
공공 데이터 및 외부 자료를 기반으로 수집되었으며,
용량 및 라이선스 이슈로 인해 GitHub에는 포함하지 않는다.

## 기상 데이터
- 출처: 기상청 (KMA)
- 내용: 시간 단위 기온, 습도, 풍속, 강수량, 일사량 등
- 활용: 체감온도, TW(습구온도) 등 파생 기상 지표 계산

## 지하철 데이터
- 출처: 서울교통공사
- 내용:
  - 역별 시간대 혼잡도 비율(%)
  - 역별 승하차 인원 데이터
- 활용: 기준 수용 인원 추정 및 혼잡도 재구성

## 외부 이벤트 데이터
- 공휴일 여부: 공공 캘린더 자료 기반 수작업 정합화
- 코로나 관련 변수: 공개 자료를 참고하여 날짜 단위로 정리

원본 데이터는 `data/processed/`에 포함된 샘플 데이터와
`preprocessing` 노트북을 통해 재현 가능한 구조로 설계되었다.
