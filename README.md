# 기상 데이터를 활용한 지하철 혼잡도 예측

## 프로젝트 개요
본 프로젝트는 기상 데이터와 과거 혼잡도 정보를 활용하여  
1시간 후 지하철 혼잡도(inner / outer)를 예측하는 것을 목표로 한다.

현실적으로 지하철 혼잡도는 정확한 정답 데이터를 얻기 어렵기 때문에,  
본 프로젝트에서는 불완전한 데이터를 어떻게 설계하고 활용할 것인가에 초점을 두었다.

---

## 문제 정의
실제 지하철 혼잡도 데이터에는 다음과 같은 한계가 존재한다.

- 혼잡도 비율(%)은 제공되지만,  
  열차 편성 수나 정확한 수용 인원 정보는 제공되지 않음
- 실시간 혼잡도 라벨을 대규모로 수집하기 어려움

이 프로젝트는 위 제약 조건을 고려하여  
현실적인 데이터 설계와 예측 파이프라인 구축을 목표로 한다.

---

## 데이터 설계
- 학습 데이터: 2021–2023년
- 검증 데이터: 2024년
- 단위: 역 × 시간

### 주요 입력 변수
- 기상 정보: 기온, 체감온도, 습도, 풍속, 일사량, TW
- 시간 정보: 시간대, 요일
- 과거 혼잡도:
  - 1시간 전 혼잡도
  - 24시간 전 혼잡도

---

## 혼잡도 산출 방식
열차 수용 인원 정보가 제공되지 않는 문제를 해결하기 위해,  
과거 승차 인원과 혼잡도 비율을 활용하여 기준 수용 인원을 역산하였다.
