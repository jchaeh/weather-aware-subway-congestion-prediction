# 기상 데이터를 활용한 지하철 혼잡도 예측

## 프로젝트 개요
본 프로젝트는 기상 데이터와 과거 혼잡도 정보를 활용하여 1시간 후 지하철 혼잡도를 예측하는 것을 목표로 한다.

현실적으로 지하철 혼잡도는 정확한 정답 데이터를 얻기 어렵기 때문에,  
본 프로젝트에서는 불완전한 데이터를 어떻게 설계하고 활용할 것인가에 초점을 두었다.

---

## 문제 정의
실제 지하철 혼잡도 데이터에는 다음과 같은 한계가 존재한다.

- 혼잡도 비율(%)은 제공되지만, 열차 편성 수나 정확한 수용 인원 정보는 제공되지 않음
- 실시간 혼잡도 라벨을 대규모로 수집하기 어려움

이 프로젝트는 위 제약 조건을 고려하여 현실적인 데이터 설계와 예측 파이프라인 구축을 목표로 한다.

---

## 데이터 설계
- 학습 데이터: 2021–2023년
- 검증 데이터: 2024년
- 단위: 역 × 시간

### 주요 입력 변수
- 기상 정보: 기온, 체감온도, 습도, 풍속, 일사량, TW
- 시간 정보: 시간대, 요일
- 과거 혼잡도:
  - 1시간 전 혼잡도
  - 24시간 전 혼잡도

---

## 혼잡도 산출 방식
열차 수용 인원 정보가 제공되지 않는 문제를 해결하기 위해, 과거 승차 인원과 혼잡도 비율을 활용하여 기준 수용 인원을 역산하였다. 

기준 수용 인원 = 실제 승차 인원 ÷ (혼잡도 / 100)

이를 통해 동일한 기준으로 혼잡도를 재구성하고, 시간대별 혼잡도 변화를 일관되게 추정할 수 있도록 설계하였다.

---

## 데이터 전처리 파이프라인
전처리 파이프라인은 다음 원칙을 기반으로 설계되었다.

- 역에 독립적인 구조
- 시계열 특성 반영
- 재현 가능성 확보

주요 처리 과정:
- 기상 데이터와 지하철 데이터 시간 단위 병합
- 시계열 기반 결측치 보정
- Min-Max Scaling
- 시계열 입력을 위한 시퀀스 생성

모든 전처리 로직은 `preprocessing/` 디렉토리에 구현되어 있다.

---

## 모델 구성
다음 세 가지 모델을 구현하고 비교하였다.

- **MLP**: 시계열 구조를 사용하지 않는 baseline 모델
- **LSTM**: 시계열 패턴 학습을 위한 순환 신경망
- **GRU**: 학습 안정성과 예측 성능을 고려한 경량 시계열 모델

모든 모델은 내선/외선 혼잡도를 동시에 예측한다.

---

## 결과 요약
- GRU 모델이 가장 높은 상관계수와 안정적인 예측 성능을 보였다.
- LSTM은 장기 패턴 반영은 가능했으나 예측 분산이 상대적으로 컸다.
- MLP는 단순 구조 대비 준수한 baseline 성능을 기록하였다.

모델별 성능 지표는 `results/metrics.csv`에서 확인할 수 있다.

---

## 활용 가능성
- 역별 단기 혼잡도 예측
- 혼잡 구간 사전 알림
- 지하철 운영 및 수요 예측 보조 지표

---

## 참고 사항
- 원본 데이터는 용량 및 라이선스 문제로 공개하지 않는다.
- 전처리 완료 데이터는 구조 확인을 위한 샘플만 제공한다.
